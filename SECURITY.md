# Security Policy

## Supported Versions

Only the latest version on the `master` branch is supported with security updates.

## Reporting a Vulnerability

If you discover a security vulnerability, please report it privately by emailing c.luiseduardom@gmail.com. Do not open a public issue.

Include as much detail as possible: steps to reproduce, affected endpoint(s), and potential impact. You should receive a response within a few days.

Please do not test for vulnerabilities against production data or attempt to access other users' data.
